# Быстрый старт

Добро пожаловать в Fogo! Это руководство поможет вам начать работу с самым быстрым блокчейном, совместимым с SVM.

## Что нужно знать

Fogo — это Layer 1 блокчейн, работающий на Solana Virtual Machine (SVM). Это означает:

* ✅ Все ваши любимые приложения Solana работают на Fogo
* ✅ Скорость выше: блоки за 40 миллисекунд, финализация за 1,3 секунды
* ✅ Простота использования благодаря Fogo Sessions — никаких постоянных подписей и комиссий

## Начало работы

### 1. Подключите кошелёк

Используйте любой кошелёк, совместимый с Solana:
* Phantom
* Solflare
* Backpack
* И другие

### 2. Получите токены $FOGO

Токен $FOGO используется для:
* Оплаты комиссий за транзакции
* Стейкинга для обеспечения безопасности сети
* Участия в управлении

### 3. Начните пользоваться приложениями

При запуске доступны следующие приложения:

**Торговля и обмен:**
* **Ambient** — перпетуальные контракты нового поколения
* **Valiant** — спотовая торговля и ликвидность
* **FluxBeam** — DEX и торговый бот для Telegram
* **Invariant** — инновационная DEX на SVM

**Кредитование:**
* **Pyron** — денежный рынок для заимствования
* **FogoLend** — платформа кредитования

**Стейкинг:**
* **Brasa** — ликвидный стейкинг (stFOGO)

**Мосты:**
* **Portal Bridge** — официальный мост для перевода активов

### 4. Используйте Fogo Sessions

Fogo Sessions делает взаимодействие с dApp таким же простым, как использование обычных веб-сайтов:

* Один раз подключите кошелёк
* Работайте со всеми приложениями без постоянных подписей
* Никаких комиссий за газ для пользователей
* Безопасность и контроль остаются у вас
